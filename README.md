# narrow-book

This is a plugin for narrow down books.
It navigates to the selected book (go to workspace).


### Install

```
ipm install narrow-book
```

## Commands

| Command          | Explanation        |
| ---------------- | -------------------|
| narrow-book:open | open book selector |

## Settings

| key            | default                     |
| ---------------| ----------------------------|
| migemoDictPath | migemo ditionary file path  |
